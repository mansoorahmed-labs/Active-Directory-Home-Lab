# Active Directory Home Lab

## Project Overview

This project demonstrates the deployment of a Windows Active Directory environment using virtualization.

The lab simulates a small enterprise network consisting of:

- Windows Server 2022
- Active Directory Domain Services
- DNS
- DHCP
- Windows 10 Client
- VirtualBox/VMware

---

## Objectives

- Install Windows Server
- Configure Active Directory Domain Services
- Promote Domain Controller
- Configure DNS
- Configure DHCP
- Create Organizational Units
- Create Domain Users
- Join Windows 10 Client to Domain
- Test Authentication
